Angular Seed Project Docs — Home

The ia-angular-seed project is an application skeleton for a typical AngularJS web application. This application attempts to adhere to the johnpapa Angular Style Guide and includes example usage of the Angular Material library, a sample service and directive, and a simple application structure that is ready to be tweaked and molded.
